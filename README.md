## ✨ 特色功能

- 🎮 **像素风格设计** - 复古游戏风格，Press Start 2P字体
- 🍶 **茅台瓶动画** - 像素化茅台瓶，浮动金币效果
- 📈 **动态K线图** - 模拟茅台股票万年长牛 + $600519币暴涨
- 🐦 **一键分享** - 集成Twitter分享功能
